Calico Cafe Website

React Cat Cafe Website

Calico Cafe Website is a site for a cat cafe built using Bootstrap and React. Features include controlled forms, modals, and navigation. The site is responsive and optimized for desktop and mobile usage.

Three pages are linked to from the navigation: a Visit page, Adopt page, and Events. The Visit page contains a form to collect information and schedule a visit time.

The Adopt page contains a listing of available cats with basic information. Clicking on a cat image launches a modal containing a larger image of the cat with additional information.
